# y72-Frontend-work

A visually immersive and performant frontend web project inspired by the official [k72.ca](https://k72.ca/) website, leveraging the power of **React**, **Vite**, **TailwindCSS**, and **GSAP** for modern, animated, and responsive user experiences.

## Project Overview

This repository delivers a robust React-based frontend, engineered for creative web portfolios, agencies, and modern business sites. It combines cutting-edge libraries and sleek design patterns to provide fast loading times, fluid navigation, and eye-catching animations.

> **Inspired by the design and interactivity of [k72.ca](https://k72.ca/).**

## Features

- **React & Vite Integration:** Efficient development and build process for modern web applications.
- **GSAP Animations:** Advanced page transitions and animated loaders for engaging user interaction.
- **TailwindCSS:** Utility-first CSS framework for rapid, responsive design customization.
- **Full-Screen Navigation:** Animated, context-aware navigation overlays for an immersive experience.
- **Project Showcases:** Visual galleries with smooth transitions and interactive effects.
- **Contextual State Management:** Navbar state managed with React Context API.
- **Video Integration:** Easily display background or hero section videos using modular components.
- **Comprehensive ESLint Configuration:** Ensures code quality and consistency throughout the project.

## Technology Stack

- **React** (JavaScript)
- **Vite**
- **TailwindCSS**
- **GSAP** & **@gsap/react** (animations)
- **React Router DOM** (routing)
- **ESLint** (linting)
- **HTML5/CSS3**

## Getting Started

To set up and run this project locally:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Kunal-Rawat1294/y72-Frontend-work.git
   cd y72-Frontend-work
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start the Development Server**
   ```bash
   npm run dev
   ```

4. **Build for Production**
   ```bash
   npm run build
   ```
